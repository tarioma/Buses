## 4. Route
Диспетчер создаёт конкретный путь (последовательность StationsEdge), по которому должно идти транспортное средство в определённый промежуток времени.

| ------------------------------------ | ----------------------------------------------- | -------- | ---- | ----------- | ------------------------------------ |
| f49f7ffd-3c3a-41c8-b68a-98b1189eff14 | Маршрут для троллейбуса №19А с нового года 2026 | 2026-1-1 | null | false       | 7cbb0d8e-ce5d-47b6-937d-af85a15ff1da |
## 5. RouteStationEdge
Диспетчер задаёт последовательность из StationsEdge для Route. Пути идут по порядку относительно Index.

| Index | RouteId                              | StationsEdgeId                       |
| ----- | ------------------------------------ | ------------------------------------ |
| 0     | f49f7ffd-3c3a-41c8-b68a-98b1189eff14 | 89fb37b6-60e4-4905-8c59-f5681445c714 |
| 1     | f49f7ffd-3c3a-41c8-b68a-98b1189eff14 | fd533547-4a19-48a7-a55b-668a3af5f948 |
| 2     | f49f7ffd-3c3a-41c8-b68a-98b1189eff14 | b93132b9-deee-4fc7-b5f0-7ac426f3a051 |
| 3     | f49f7ffd-3c3a-41c8-b68a-98b1189eff14 | 0b35c3ea-5756-43b3-9be1-e0f3efa31d96 |

Теперь у диспетчера есть путь, по которому должен идти троллейбус и время, за которое он должен этот путь пройти: 4 + 3 + 3 + 6 = 16 минут.

## 7. VehicleGlobalRoute
Диспетчер задаёт, по какому GlobalRoute должно двигаться Vehicle в определённый диапазон дат. Также, со скольки и до скольки по времени.

| Id                                   | GlobalRouteId                        | VehicleId                            | From  | To    | DayOfWeek | StartDate | EndDate |
| ------------------------------------ | ------------------------------------ | ------------------------------------ | ----- | ----- | --------- | --------- | ------- |
| bccd41db-6056-4a49-a3ea-e13899026984 | 7cbb0d8e-ce5d-47b6-937d-af85a15ff1da | 4c727180-86bf-498d-96c2-d9d1a6c6c485 | 8:00  | 12:00 | Monday    | 2026-1-1  | null    |
| 58d10b80-8a2b-4a44-a244-c3c1ac933738 | 841391d4-b0b3-41fe-bc40-8c77cd2e8e14 | 4c727180-86bf-498d-96c2-d9d1a6c6c485 | 13:00 | 20:00 | Monday    | 2026-1-1  | null    |

Таким образом троллейбус с номером T123AA по понедельникам с 8 утра до 12 дня будет ездить по маршруту №19А, а с 13:00 до 20:00 по маршруту №2.
Таблицы заполнил данными только для №19А. Для корректной работы системы подразумевается, что №2 тоже должен быть заполнен.
